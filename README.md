8-Puzzle Solver with PyQt5
8-Puzzle Icon
تطبيق رسومي تفاعلي لحل لغز الـ 8 (8-Puzzle) باستخدام خوارزميات البحث الذكي في الذكاء الاصطناعي
هذا البرنامج يتيح لك إدخال حالة أولية للغز الـ 8، ثم مشاهدة عملية الحل خطوة بخطوة باستخدام إحدى الخوارزميات الثلاث، مع أنيميشن بصري رائع لتحرك البلاطات على اللوحة.
المميزات الرئيسية

واجهة رسومية جميلة باستخدام PyQt5
ألوان مميزة لكل رقم من 1 إلى 8 (مع مساحة فارغة بيضاء)
إدخال الحالة الأولية يدويًا عبر مربع نص (مثال: 1 2 3 4 5 6 7 8 0)
فحص تلقائي لإمكانية الحل (بناءً على عدد الانعكاسات - Inversion Count)
دعم ثلاث خوارزميات بحث:
BFS (Breadth-First Search) → يضمن أقصر مسار لكن بطيء في بعض الحالات
DFS (Depth-First Search) → سريع لكن قد لا يجد أقصر مسار
A* (A-Star) → الأفضل عمليًا (يستخدم Manhattan Distance كـ heuristic)

أنيميشن لحركة البلاطات أثناء عملية البحث
أزرار تحكم كاملة:
Start لبدء الحل
Stop لإيقاف العملية في أي وقت
Replay لإعادة تشغيل المسار الأمثل ببطء (للمشاهدة التعليمية)

عرض طول المسار الأمثل عند الانتهاء

مثال على إدخال حالة قابلة للحل
text2 5 3 1 6 4 7 0 8
(اكتب الأرقام مفصولة بمسافات في المربع، واستخدم 0 للمكان الفارغ)
طريقة التشغيل

تأكد من تثبيت PyQt5:Bashpip install PyQt5
ضع صورة الأيقونة IdlibIcon.png في نفس مجلد الكود (اختياري، لكن يعطي شكلًا احترافيًا).
شغّل البرنامج:Bashpython puzzle_solver.py

كيفية الاستخدام

اكتب الحالة الأولية في المربع (9 أرقام من 0 إلى 8 بدون تكرار).
اضغط Read Input لعرض اللوحة.
اختر الخوارزمية المطلوبة (A* موصى بها).
اضغط Start لمشاهدة عملية الحل خطوة بخطوة.
استخدم Stop إذا أردت إيقاف البحث.
بعد إيجاد الحل، اضغط Replay لمشاهدة المسار الأمثل ببطء مع رسالة توضح طول المسار.

ملاحظة: إذا ظهرت رسالة "The state is not solvable"، فهذا يعني أن الحالة غير قابلة للحل رياضيًا (نصف حالات الـ 8-puzzle غير قابلة للحل).
